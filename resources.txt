AUTENTICAÇÃO

## Register
- auth/signup
- Name
- Email
- Password
(JSON, POST)
-- devolver um token de autenticação junto com os dados do usuário
{
    user: {"id":1, "name": "Renato", "email": "renato@teste.com.br"},
    token: "ljdaifujclskfhneeiurflks"
}

## Login
- auth/signin
- Email
- Password
(JSON, POST)
- verificar se o email e a senha estão corretos, se sim, retornar
{
    user: {"id":1, "name": "Renato", "email": "renato@teste.com.br"},
    token: "ljdaifujclskfhneeiurflks"
}

## Verify - Página de verificação. Retorna os dados do usuário
- auth/verify
- token
(POST, Cabeçalho da requisição)
Authorization: bearer <token>
{
    user: {"id":1, "name": "Renato", "email": "renato@teste.com.br"},
    token: "ljdaifujclskfhneeiurflks"
}
